Puntos de venta por usuarios

Permitir acceso a usuarios solo en los TPV configurados para cada usuario(escenario multi tiendas)

Principales Funcionalidades
  • Configurar en cada usuario los TPV a los que tiene acceso
  • Mostrar solo los TPV que el usuario tenga permiso
  • En el TPV, solo cargar los vendedores de ese TPV
