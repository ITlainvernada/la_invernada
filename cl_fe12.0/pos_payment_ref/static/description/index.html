Codigo de autorizacion en pagos tipo banco

Permite configurar diarios en los que se pide un codigo de autorizacion, especialmente los pagos tipo banco

Principales Funcionalidades
  • Configurar en los diarios si se exige o no un codigo de autorizacion del pago
  • Exigir ingresar el codigo de autorizacion, no permite validar la venta si no se ingresa el codigo
  • Imprimir el codigo de autorizacion en el recibo de venta
