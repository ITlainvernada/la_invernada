Cash Register On Payments

This module added features on customer/supplier payments to allow account user to link payment with cash register direct through payment menu or customer/supplier invoices register payment button. After selecting and validating payment, module will add cash register line on selected cash register.
Features are:

  • User can select cash register through Payments menus.
  • User can select cash register while they are on customer invoices menu and make payment through select register payment option.

Fields on Payment Form

Selected Fields on Payment Form

  • When User click on Add Cash Register Entry check box and make it ticked at that time Cash Register dropdown box will be appear other wise invisible.
  • User can select cash register form this dropdown, where user want to make entry in selected cash register.

Register Payment Screen

  • User can register payment of those customer/supplier whose invoices are validated.
  • This registration payment transaction entry are added in selected cash register if it has been selected.

Showing Entry On Cash Register

  • It's show each transaction entry which is done by payment entry or register payment option from customer invoices option when user can validate customer/supplier invoices.
